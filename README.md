# 📄 Resume Analyzer with AI (Gemini)

A full-stack web application that helps users compare their resume with a job description to identify skill gaps and prepare better for job roles.

---

## 🚀 Overview

This project allows users to upload their resume and a job description. The system analyzes both and highlights missing skills, gaps, and key differences. It also generates useful questions based on the resume and job role to help with interview preparation.

The goal is to make job preparation more focused and practical.

---

## ✨ Features

- Upload resume and job description  
- Compare both to identify skill gaps  
- Generate interview questions based on the resume  
- Secure user authentication  
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack

**Frontend:**  
- React  
- Tailwind CSS  

**Backend:**  
- Node.js  
- Express.js  

**Database:**  
- MongoDB  

**Other:**  
- Gemini API (AI-based analysis)  
- JWT Authentication  

---

## 🔐 Authentication

- User signup and login  
- JWT-based authentication  
- Protected routes  

---

## ⚙️ How It Works

1. User logs in  
2. Uploads resume and job description  
3. System analyzes both inputs  
4. Displays:
   - Skill gaps  
   - Missing keywords  
   - Suggested interview questions
